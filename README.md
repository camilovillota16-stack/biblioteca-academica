# Sistema de Gestión y Consulta de Biblioteca Académica

Proyecto final de la asignatura **Estructuras de Datos y Algoritmos 1**.

El sistema permite gestionar un catálogo de libros académicos mediante una aplicación web construida con **React** en el frontend y **FastAPI** en el backend.

## Objetivo del proyecto

Desarrollar una aplicación web funcional que permita registrar, consultar, buscar, ordenar y explorar libros académicos, aplicando conceptos de estructuras de datos, algoritmos de búsqueda, algoritmos de ordenamiento, recursión y análisis de complejidad.

## Tecnologías utilizadas

- React
- Vite
- FastAPI
- Python
- JSON como mecanismo de persistencia
- Git y GitHub
